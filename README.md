# TwitchMarkov
A Markov chain generator for Twitch.

### Setup
- Install dependencies
> pip install markovify

> pip install emoji
- Make a Twitch Account
- Generate an oAuth token with https://twitchapps.com/tmi/
- Fill out the conf.py file.
- Execute.
